# Bill Splitter 💸

A beginner Python project that calculates the total restaurant bill, adds a 25% tip, and splits the final amount equally among friends.

## Features

- Calculates the total food and drink cost
- Adds a 25% tip automatically
- Splits the final bill equally among friends
- Rounds the amount each person pays to two decimal places

## Concepts Used

- Variables
- Arithmetic Operators
- Assignment Operators (`+=`)
- `print()`
- `round()`
- Basic Calculations

## Example Output

```text
Total bill so far: 198.83
Tip amount: 49.7075
Total with tip: 248.5375
Bill per person: 62.134375
Each person pays: 62.13
```
