# 💸 Apply Discount Function

A simple Python project that calculates the final price of an item after applying a percentage discount.

## 📖 About

This project was built to practice Python functions, input validation, and conditional statements. It checks whether the provided inputs are valid before calculating the discounted price.

## ✨ Features

- Validates the price and discount inputs
- Handles invalid data types
- Ensures the price is greater than 0
- Ensures the discount is between 0 and 100
- Calculates the final price after applying the discount

## ▶️ Example

**Input**

```python
print(apply_discount(100, 20))
```

**Output**

```
80.0
```
