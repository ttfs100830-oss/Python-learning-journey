# 🧭 Travel Weather Planner

A Python project that determines whether commuting is possible based on travel distance, weather conditions, and available transportation.

## 📖 About

This project was built to practice **conditional statements** and **Boolean logic** in Python. It evaluates different travel scenarios and decides whether commuting is possible using `if`, `elif`, and `else` statements.

## 🚀 Features

- Checks travel distance
- Considers rainy weather
- Supports multiple transportation options:
  - 🚲 Bike
  - 🚗 Car
  - 🚕 Ride-share app
- Uses Boolean operators (`and`, `or`, `not`) to make decisions

## ▶️ Example

**Input**
```python
distance_mi = 8
is_raining = True
has_bike = True
has_car = False
has_ride_share_app = True
```

**Output**
```
True
```
