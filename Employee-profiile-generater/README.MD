# Employee Profile Generator

A beginner-friendly Python project that generates an employee profile and demonstrates Python fundamentals

## Features
- String concatenation
- f-strings
- String slicing
- Formatted employee card

## Concepts Used
- Variables
- Strings
- Data types
- String indexing
- String slicing

## SAMPLE OUTPUT

Employee: John Doe
Age: 28
Position: Data Analyst
Salary: $75000

Department: DEV
Year: 2026
Initials: JD
Employee Number: 001
